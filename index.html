推しくらべ

OSHI CHECK

対面で、推しYouTubeチャンネルの「かぶり」と相性度を
その場で答え合わせするiOSアプリ。

🔒 一致した推しだけを開示 — 非一致はお互いに見えません

使い方は3ステップ

1

推しを取り込む

Googleアカウントでログインして、YouTubeの登録チャンネルを取り込みます。取り込んだ一覧は自分の端末にだけ保存され、共有していい推しだけを自分でオンにします。秘密にしたい推しはオフのままでOK。

2

相手とつなぐ

2人とも「相手とつなぐ」を押すだけ。その場の端末どうしが直接通信します（照合内容がサーバーを通ることはありません）。3人以上の「みんなでつなぐ」もあります。

3

結果を同時に見る

ふたりの画面に同時に結果が開示されます。かぶった推しチャンネル、共通ジャンル、そして「推しかぶり度」。紙吹雪と一緒に盛り上がってください。

プライバシーの仕組み

  • 一致した推しだけがお互いの画面に表示されます。
  • 一致しなかった推し・登録リストの中身は、暗号学的手法（PSI: Private Set Intersection）により相手にも運営にも渡りません。
  • 照合はその場の端末どうしの直接通信。登録チャンネルの一覧が外部サーバーに送信・保存されることはありません。
  • 取り込んだ一覧（カタログ）は端末内のみに保存され、30日で自動的に消去されます。

詳しくは プライバシーポリシー をご覧ください。

本アプリは YouTube API Services を使用しています。ご利用にあたっては YouTube 利用規約 および Google プライバシーポリシー が適用されます。アプリに許可したアクセス権は Googleアカウントのセキュリティ設定 からいつでも取り消せます。
