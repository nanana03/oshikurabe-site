プライバシーポリシー

推しくらべ (Oshi Check)

本ポリシーは、iOSアプリ「推しくらべ」（英語名: Oshi Check、以下「本アプリ」）における利用者情報の取り扱いを定めるものです。

1. 取得する情報

本アプリは、利用者の許可（Googleアカウントによるログインと同意）に基づき、YouTube Data API を通じて次の情報を取得します。

  • YouTubeの登録チャンネル一覧（チャンネル名・チャンネルID・アイコン画像・チャンネルの公開情報〔説明文・キーワード・トピックカテゴリ〕）— 読み取り専用スコープのみを使用し、アカウントの他の情報には一切アクセスしません。

本アプリが独自に氏名・メールアドレス・位置情報などを取得することはありません。

2. 利用目的

  • 取り込んだチャンネル一覧を端末内に表示し、利用者が「共有してよい推し」を選択できるようにするため
  • チャンネルをジャンルへ自動分類し、一覧を見やすくグループ表示するため
  • 対面の相手との「共通の推し・共通ジャンル・相性度」の照合を行うため

3. 保存場所と保持期間

  • 取り込んだチャンネル一覧（カタログ）は利用者の端末内にのみ保存されます。運営者のサーバーには保存されません。
  • カタログは取り込みから30日で自動的に消去されます（再取り込みで更新できます）。
  • 利用者が共有用に選択したチャンネルのリストも端末内にのみ保存されます。
  • アプリを削除すると、端末内のすべてのデータが消去されます。

4. 外部への送信

ジャンル分類のための送信

チャンネルのジャンル自動分類のため、チャンネルの公開情報のみ（チャンネルID・チャンネル名・説明文・キーワード・トピックカテゴリ）を運営者の分類サーバー（Google Cloud 東京リージョン）に送信し、分類結果を得ます。分類結果は同一チャンネルの再分類を避けるためサーバー側にキャッシュされます。誰がどのチャンネルを登録しているかという情報（アカウント情報・登録一覧としてのまとまり）は送信・保存されません。

対面照合時の通信

相手との照合は、その場の端末どうしの直接通信（ローカルネットワーク／近接通信）で行われ、照合内容が運営者を含む外部サーバーを経由することはありません。照合には暗号学的手法（PSI: Private Set Intersection）を用いており、一致した項目だけが相互に開示され、一致しなかった項目は相手にも知られません。

5. 第三者提供・広告

  • 利用者の情報を第三者に提供・販売することはありません。
  • 本アプリに広告は表示されず、広告目的のトラッキングも行いません。

6. Google ユーザーデータの取り扱い（Limited Use）

本アプリによる Google API から取得した情報の利用は、Google API Services User Data Policy（Limited Use の要件を含む）に準拠します。取得したデータは上記第2条の機能提供のためにのみ使用し、それ以外の目的（広告・売却・信用判断等）には使用しません。

7. YouTube API Services

本アプリは YouTube API Services を使用しています。利用者には YouTube 利用規約 および Google プライバシーポリシー が適用されます。

本アプリに許可したアクセス権は、Googleアカウントのセキュリティ設定ページ からいつでも取り消すことができます。

8. ポリシーの変更

本ポリシーを変更する場合は、本ページで告知します。重要な変更がある場合は、アプリ内でもお知らせします。

9. お問い合わせ

本ポリシーに関するお問い合わせは nanana03info@gmail.com までお願いします。
